About Sneaker Spot

Sneaker Spot is your go-to destination for the latest and most stylish footwear. We specialize in curating premium sneakers from top brands like Nike, Adidas, and more. Our mission is to provide sneaker enthusiasts with a seamless shopping experience and a wide variety of options to express their style and individuality. Whether you're a casual wearer or a collector, we've got the perfect pair for you!

SneakerSpot - Angular Project

SneakerSpot is an Angular-based web application created for the project defense of the SoftUni Angular course.

Features

  • Sneaker Listings: Browse through an extensive collection of sneakers with detailed descriptions and images.
  • User Authentication: Secure login and registration system using JWT (JSON Web Tokens).
  • Favorites & Reviews: Allow users to save favorite sneakers and leave reviews.

Tech Stack

  • Frontend: Angular, TypeScript
  • Backend: Node.js, Express.js
  • Database: MongoDB
